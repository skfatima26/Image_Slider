A responsive & Interactive image slider built using HTML < CSS < JAVASCRIPT.
This slider allows user to navigate through images smoothly , providing manual controls.
Manual Controls/navigation using prev & next buttons
Full responsive for all screen sizes
Smooth transition effects

PROJECT STRUCTURE
//Image-Slider 
-/images            # image assets
-/index.html        # Main html file + Js logic
-/style.css         # Styling for slider
-/readme.md         # about project


1] Open index.html in a browser
Click the next or previous buttons to switch images
